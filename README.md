# Mac, Duro de Matar

Pagina estatica com tema de trailer/cartaz de filme de acao para o Ricardo Macarico.

## Live

- Pagina publica: `https://tjcorreia.github.io/mac-duro-de-matar/`
- QR code: `assets/qr-mac-d.png`

## Assets

- `assets/ricardo-macarico.mp4`: video em loop no centro da pagina
- `assets/qr-mac-d.png`: QR code com etiqueta `MAC-D`

## Nota

O video foi usado diretamente em `video autoplay loop muted playsinline`, porque neste ambiente nao tenho `ffmpeg` para converter o MP4 para GIF. Visualmente faz o mesmo efeito, com melhor qualidade e ficheiro mais leve.
